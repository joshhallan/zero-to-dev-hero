# Welcome
Hello :) welcome to the internal React training course from Leighton

This course is geared towards getting someone from knowing 0 about development to creating their own fully functioning React application. If you do run into any trouble please feel free to reach out to anyone in the FE community and I'm sure they'll be happy to help. 

This course is going to entwine with the course within the AWS community as well but that won't be for a while so don't worry!

## Structure
The course is going to be split into 3 different levels of difficult from introductory where we'll explain basic markup up to advanced which will cover advanced hooks, api calls and much more!

### Introductory
1. Project set up
    - Node version
    - Project creation
    - Explain the different parts of the project 
2. Intro into JSX
    - Discuss JSX
	- Create simple heading
	- Use js to make it a bit interactive
	- Restyle using css
3. Reusable components
	- Create global component
	- Style it
	- Pull it into the App.tsx
4. Props
    - Explain what props are
	- Add variant styling for the button

### Intermediate
1. Flat data
2. explain state / hooks
3. authentication / auth
4. use state
5. routing
6. form management
7. mapping
